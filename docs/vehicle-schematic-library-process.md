# Vehicle Schematic Library Process

## Locked Process

Vehicle schematics must be built from real reference geometry before being added to the app.

1. Confirm the exact vehicle type, generation, and body configuration.
2. Gather real reference views for left, right, front, and rear.
3. Include only practical inspection geometry: body panels, doors, windows, wheel arches, lights, bumpers, mirrors, side steps, tray or body structure, and emergency roof lights where relevant.
4. Exclude all words, badges, logos, branding, number plates, people, decorative graphics, and emergency-service symbols.
5. Produce a clean damage-marking schematic with four views only: Left, Right, Front, Rear.
6. Show the schematic preview before hard-coding it into the app.
